The following animations summarize the molecular dynamics simulations run with the Jones potential and Verlet integration. For illustrative aid, the periodic domain has been tiled around the blue square. That is, all position, velocity, energy, and pressure calculations are done within the blue domain, and the rest of the circles are added for visual effect. Some simulations are run backwards and are indicated by color (red).
